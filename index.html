📦 資料匯出 / 匯入
在不同瀏覽器或裝置間搬移任務記錄。從舊入口「匯出」，再到新入口「匯入合併」即可。
📤 匯出：點擊後會把所有記錄顯示在上方文字框，全選複製即可。
📥 匯入合併：將複製的 JSON 貼入上方文字框，點擊後會合併（不會覆蓋已有的記錄）。
📋 Make.com Webhook 設定說明
在主頁貼上你的 Make.com Webhook 網址。任務結束或編輯後，App 會自動把資料送到這個網址，由 Make.com 負責寫入 Google Calendar。
🛠 Make.com 設定步驟
  • 1
    到 make.com 註冊／登入，新增 Scenario
  • 2
    第一個模組選 Webhooks → Custom webhook，複製網址
  • 3
    第二個模組選 Google Calendar → Create an Event
  • 4
    用個人 Gmail 授權 Google Calendar（不用 Workspace 帳號）
  • 5
    欄位對應：Summary = [cat] task，Start = start，End = end
  • 6
    開啟 Scenario（打開開關），把 Webhook 網址貼到主頁儲存
📅 行事曆 API 設定步驟
  • 1
    到 script.google.com 新增專案，貼上提供的 GAS 程式碼
  • 2
    修改 CONFIG 區的分頁名稱，對應你的實際 Sheet Tab 名稱
  • 3
    「部署」→「新增部署」→ 類型選「網頁應用程式」
  • 4
    執行身份選「我自己」，誰可存取選「所有人」
  • 5
    複製部署網址，貼到行事曆頁面的 API 輸入框中
  • 6
    先在 GAS 編輯器執行 testGetAll() 測試是否正常
